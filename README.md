# GreenScape Landscaping

A professional landscaping business website built with React and TypeScript to practice modern web development and automated testing.

## Features
- Multi-page navigation with React Router
- Responsive navbar and layout
- Services showcase page
- Project gallery
- Contact form with validation
- Comprehensive Selenium test suite

## Tech Stack
- React
- TypeScript
- React Router
- Selenium WebDriver (automated testing)

## Purpose
Built as a learning project to practice React fundamentals, routing, component architecture, and automated front-end testing with Selenium.
